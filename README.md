# ToxiGuard

**Aide décisionnelle en toxicologie d'urgence pour médecins.**

Outil web gratuit d'aide à la décision pour la prise en charge des intoxications
aiguës en médecine d'urgence. Application monopage (un seul fichier `index.html`),
sans dépendance ni serveur : elle fonctionne entièrement dans le navigateur.

## Avertissement médical

Outil pédagogique d'aide à la décision destiné aux professionnels de santé.
Il ne remplace ni l'évaluation clinique, ni l'avis du Centre Antipoison (CAPTV),
ni les recommandations officielles. L'utilisateur reste seul responsable de toute
décision diagnostique ou thérapeutique.

## Mise en ligne (Vercel)

Le projet est un site statique. Pour le déployer :

1. Pousser ce dépôt sur GitHub.
2. Sur [vercel.com](https://vercel.com) : **Add New… → Project → Import** ce dépôt.
3. Laisser les réglages par défaut (Framework : *Other*) puis **Deploy**.

Toute modification poussée sur la branche principale redéploie automatiquement le site.

## Mettre à jour l'application

Remplacer le contenu de `index.html` dans le dépôt GitHub. Vercel redéploie tout seul.
